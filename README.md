# Multi Pages Webpack
- Each html should have its corresonding `bundle.js`. 

## How to start?
1. Create `webpack.config.js`, and edit it. 
2. Initialise npm. `npm init`
3. Edit `package.json`. 
4. Install following packages. 
    ```
    npm install --save-dev webpack webpack-cli lite-server
    ```
